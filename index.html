Computer Science

About Computer Science


Taught by Mr. Orkney

The goal of this class is to teach students the importance of computer science, and the basic principles of coding. Students will learn many different coding languages, and will learn how computer science has developed over time.

Code languages include but not limited to: php, html, python, and more.

Examples of skills learned: Logic statements, classes, loops, assigning variables, string definitions, etc.

Summary of Tools


Web Programming

  • HTML: basic language used by web browsers to display websites.
  • CSS: language used to "beautify" your websites.
  • PHP: server side code used to generate dynamic website code.

Other

  • Python: coding language that can be proccessed by machines, or small projects. Uses: robots, single use devices, and many others.
  • RobotC: modified C language meant for use with Vexnet robots. (May not be continued in the curriculum)
